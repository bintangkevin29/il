## Installation

Or if you want to run this project locally, in the project directory, you can run:

```
yarn install
```

```
yarn start
```

#### Recommended System Requirements
- Node v16.*
- yarn v1.22.*

## Run

And then in your browser, visit

```
http://localhost:3000/
```
